If you have correctly unpacked the Unix
tarred and gzipped distribution file,
you should have the following files in
your working directory:

aqplot.f
aqplot.in
asf1101559.aq
asfplot.ps
asi1101559.aq
asiplot.ps
asmanual.pdf
Makefile
README.txt
tcas.cmd
tcas.f
ts1101559.aq
ts1101559.ttr

Detailed instructions on the use of the
adaptive stacking program TCAS are 
contained in the PDF manual "asmanual.pdf".
